## Building from Source for Developers

```console
$ git clone https://github.com/TralahM/TCPIPSocketsInC.git
$ cd TCPIPSocketsInC
$ mkdir build
$ cd build
$ cmake ../src
$ make
# Or install to a desired prefix
$ make install
```
